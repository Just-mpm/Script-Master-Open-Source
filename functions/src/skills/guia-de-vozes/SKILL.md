# Guia de Vozes — Gemini TTS

O Script Master usa o modelo Gemini 3.1 Flash TTS com **30 vozes pré-definidas**. Cada voz tem um estilo natural e responde bem a direções de prompt. Pense no modelo como um **ator virtual**: a voz escolhida define o timbre, e as instruções do prompt definem a interpretação.

## Como Escolher a Voz Certa

1. **Combine voz com o conteúdo** — uma voz animada para roteiro energético, uma voz suave para narração intimista
2. **Alinhe voz com a emoção** — não peça para uma voz grave soar como uma criança; o resultado será inconsistente
3. **Teste antes de commitir** — use o preview de voz no estúdio para ouvir como soa com o roteiro

## Multi-locutor (Diálogos)

O sistema suporta **2 vozes simultâneas**. Para diálogos:

- **Voz A** (locutor principal): escolha baseada no papel principal
- **Voz B** (locutor secundário): escolha complementar — contraste de estilos funciona bem

### Combinações Recomendadas

| Tipo de Diálogo | Voz A | Voz B | Por quê |
|-----------------|-------|-------|---------|
| Entrevista | Charon (informativa) | Aoede (descontraída) | Jornalista + convidado relaxado |
| Podcast casual | Zubenelgenubi (casual) | Achird (amigável) | Conversa entre amigos |
| Aula/dialogada | Sadaltager (especialista) | Leda (jovem) | Professor + aluno |
| Dramático | Gacrux (madura) | Enceladus (aérea) | Contraste grave × suave |
| Noticiário | Charon (informativa) | Kore (firme) | Dupla de ancoragem |

## Dicas Rápidas

- **Roteiro de marketing?** → Pulcherrima (direta) ou Puck (animada)
- **Narração de documentário?** → Charon (informativa) ou Rasalgethi (informativa)
- **Meditação/bem-estar?** → Umbriel (tranquila) ou Vindemiatrix (gentil)
- **Aula/explicação?** → Sadaltager (especialista) ou Iapetus (clara)
- **Storytime/infantil?** → Leda (jovem) ou Achernar (macia)
- **Podcast energético?** → Zephyr (brilhante) ou Aoede (descontraída)
